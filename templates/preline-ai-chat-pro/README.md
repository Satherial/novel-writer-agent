# Preline — AI Chat Pro (riferimento template)

Questa cartella **non** vendora il template proprietario di Preline nel repository (licenza / aggiornamenti).

Per il layout “AI Chat Pro” ufficiale:

- Documentazione e snippet: [Preline UI](https://preline.co/)
- Template Pro (se acquistato/abbonamento): sezione template sul sito Preline

L’app usa **Preline JS/CSS v4** tramite il pacchetto `preline` (vedi `globals.css` + `preline-provider.tsx`) e componenti Tailwind allineati alla dashboard e alla pagina progetto.
